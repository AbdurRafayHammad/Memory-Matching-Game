# Memory Matching Game

Memory Matching Game is a simple browser-based game built using HTML, CSS, JavaScript, and jQuery. The objective of the game is to flip cards and find matching pairs. The game tracks the player's moves, time, and progress while providing visual animations and sound effects for a better user experience.

# Features

16 card grid (4×4) memory board
Card flip animation using CSS 3D transforms
Matching system that detects correct and incorrect pairs
Live timer that starts on the first move
Move counter to track player attempts
Progress bar showing completion status
Win screen overlay when all pairs are matched
Sound effects using the Web Audio API
Modern responsive user interface

# Technologies Used

HTML5 for the structure of the web page
CSS3 for styling, layout, and animations
JavaScript for implementing the game logic
jQuery for DOM manipulation and event handling
Web Audio API for generating sound effects

# How to Run the Project

First clone the repository

git clone https://github.com/your-username/memory-matching-game.git

Then open the project folder and run the file named index.html in any web browser.

No installation or additional setup is required.

# How to Play

Click on a card to flip it.
Flip another card to try to match the pair.
If both cards match, they remain open.
If they do not match, they flip back.
Continue playing until all pairs are matched.

The game records the total number of moves and the time taken to complete the game. Once all pairs are found, a win screen appears showing the final statistics.
